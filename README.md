## Installation
Make sure you're in the correct directory before attempting to start the app.

```
$ cd app/
```

To install the dependencies:

```
$ pip install -r requirements.txt
````
To start the server

```
$ python3 app.py
```

To populate the database
```
$ python3 -i src/db.py
>>> create_tables()
```

Test at:
```
localhost:8080/
```

Default credentials:
```
username: a
password: a
```

To debug:
```
$ python3 -i debug.py
```
